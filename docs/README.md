# Документация jai

## Проектные документы (фаза проектирования)
- **MVP_руководство_процесс_Linux_Windows.html** — пошаговое руководство по разработке (11 фаз, 42 шага).
- **Инженерный_отчёт_финальный.html** — инженерный отчёт: ключевые решения и набор моделей.
- **jai_дизайн_макет.html** — HTML-прототип интерфейса.

Это снимки фазы проектирования (открываются двойным кликом в браузере) — обоснование
решений и план. **Актуальное состояние реализации** см. в журнале изменений
[`../CHANGELOG.md`](../CHANGELOG.md).

## Отчёты по этапам
