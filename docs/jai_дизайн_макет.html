Рассуждение · 4 сек
Нужно соединить таблицы клиентов и заказов, отфильтровать по городу «Москва» и по дате за последние 30 дней. Чтобы клиент не повторялся при нескольких заказах — DISTINCT. Сортировка по имени для удобства чтения.

Вот запрос — уникальные клиенты из Москвы с заказом за последние 30 дней:

sql 
SELECT DISTINCT c.id, c.name, c.city
FROM clients c
JOIN orders o ON o.client_id = c.id
WHERE c.city = 'Москва'
  AND o.created_at >= date('now', '-30 days')
ORDER BY c.name;
